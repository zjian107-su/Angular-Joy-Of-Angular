RXJS

Description

RXJS topic includes observables, subjects, and operators. Observables are the foundation of RXJS. Subjects are a type of observable that allows values to be multicasted to many observers. Operators are functions that build on the observables foundation to enable sophisticated manipulation of collections.

Observable vs promises. Observable, Observer, MultiCasting, Subject, Unique such as distinctUntilChanged(). These are some of the concepts that are used in NGRX. NGRX is a powerful library that helps manage the state of the application.
